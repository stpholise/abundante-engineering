
# AbundantEngineering

 AbundantEngineering is a frontend website built with Next.js, utilizing Server-Side Rendering (SSR) for fetching projects and Sanity Studio for content management.

 ## Overview
 
 AbundantEngineering is a portfolio website showcasing various engineering projects, services and updates.
 This site is designed to be fast, scalable, and easy to maintain.

 ## Technologies Used

 - Next.js
 - Server-Side Rendering (SSR)
 - Sanity Studio 
 - React
 - TailwindCSS


 ## Key Features
 - Project showcase with filtering option
 - Project details page with Sanity Studio managed content
 - Responsive design for mobile and desktop devices
 - News showcase with filtering option
 - Functional quote request form

 ### Implementation

 - SSR is used to fetch project data on the server-side, improving the SEO and page load time.
 - Sanity Studio is used for content management, allowing easy updates to project information

 ## Setup Instructions 

 1. Clone the repository: git clone https://github.com/stpholise/abundante-engineering.git

 2. Install dependencies: npm install

 3. Start the develpement server: npm run dev
